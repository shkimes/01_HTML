DOM(Document Object Model)

        웹 문서(HTML)의 모든 요소에 Java Script가 접근하기 위해 설장한 방식

        요소 : <태그>내용물</태그>

        특정 태그 내에 작성된 아이디 값이나 클래스갑으로 접근할 수 있다.
        ex. document.getElementById("id속성명");
