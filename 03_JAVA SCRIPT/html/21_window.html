Window 내장 객체

        -인터넷 창 자체를 나타내는 객체

        -자바스크립트 모든 기능에서 가장 최우선으로 작동

        -window의 자식(아래)객체
            1.DOM(Document Object Model) : Document
                (브라우저에서 내용(페이지) 출력 부분)

윈도우의 기능임을 생략해도 되는 기능들

        (window.)alert("글자")
        (window.)prompt("글자")
        (window.)setTimeout("함수, 지연시간")

앞에 윈도우를 붙여줘야 하는 기능들

        window.location.href = "이동할.html"
        //window를 생략할 수 있긴하지만 상황에 따라 파일을 가져오는 위치가 변경될 수 있기 때문에 명확한 위치표기를 위해 window를 붙여주는것이 좋음
        //location = 위치 지역
★href와 src의 차이
        링크나 리소스를 참조할 때 사용하지만 사용용도가 다르다

        [href] = 페이지 이동이나 기능을 참조할 때 사용
                css파일, 페이지 간 이동, 외부 웹 페이지 링크 등..
        [src]  = 페이지 안에 이미지, 동영상, JS파일 등 외부 소스를 삽입할 때 사용
                js파일, 이미지, 동영상 등..
