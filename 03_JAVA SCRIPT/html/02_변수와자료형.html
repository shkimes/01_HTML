자바스크립트의 변수(Variable)

        *메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간
        *변수에 이름을 붙이고 원하는 값을 대입하면 필요한 것에서 변수명을 호출하여 저장된 값을 사용할 수 있다

변수이름(=변수명)명명규칙

        -의미있는 이름을 작성(대충 a,aaa,aa 이런식으로 짓는것)
        -대,소문자 구분
        -특수문자는 _(언더스코어)사용가능(단, 첫글자제외)
        -숫자 사용 가능 (단,첫글자 제외)
        -카멜표기법 작성

        [카멜 표기법(Camel Case)]
        문장, 여러 단어를 변수명을 작성할 때 이어지는 첫 번째 단어를 대문자로 작성
        ex. user name = username

        만약에 변수명 짓는데 어려움이 있다면
        https://www.curiostore.com 사이트에서 변수명 짓는것을 도움 받을 수 있음

JavaScript 변수의 종류와 특징

        변수이름을 새로 만들어서 변수가 생기는 것을 담을 공간 생성
        새로 만들었다! 를 어떻게 표기할 것인가?

        NEW 바구니 A = 물건이나 숫자가 담겼습니다;

        변수 = Variable
        var 바구니 a = '물건이나 숫자가 담겼다';

        var : 변수, 변수명 중복o (덮어쓰기가 가능)
        덮어쓰기나 여러가지 문제가 생겨서 변수명 중복x
        iet : 변수, 변수명 중복x

        const : 상수, 변수명 중복x 값 변경도 안됨

        변수 : var, let
        상수 : const (최신트랜드, 제일 많이 사용)

        1순위 : const (JS는 특정 요소를 선택해서 사용하는 경우가 많아 변수에 고정시켜둠)
        2순위 : let
        3순위 : var

자바스크립트의 자료형

        자바스크립트는 변수 선언 시 별도의 자료형을 지정하지 않음
        -> 변수에 대입되는 값(리터럴)에 의해서 자료형이 결정됨.

        -숫자는 '',"" 없이 표기 가능
        -문자열 '',""로 감싸서 작성 (영어 포함 모든 언어는 감싸야한다)
        -진실 혹은 거짓 (true false) '',""없이 표기 가능
        -function(함수, 기능)
        -undefined(정의되지 않은 변수)->변수가 선언만 되었을 뿐 값이 대입되지 않음

        (참고)
        -null (참조하는게 없음)
            ->document.getElementById("오타")
            ->id가 일치하는 요소가 없어서 참조할 수 없다.
