DOM(Document ObJect Model)

        웹 문서 (HTML)의 모든 요소를 객체 형식으로 표현한 것
        (HTML 요소 == JS에서 객체라고 부름)

        -> 문서 내 특정 요소 접근 가능
            + 요소(객체)에 속성 값을 세팅 또는 얻어올 수 있음
            (getElement = 요소)
            document.getElementById("아이디값");
            document.       = html 코드(문서) 
            .               = html코드 문서내에서 
            getElementById  = 요소들 중에서 갖고 오고 싶은 아이디 값을 가져오겠다
            ("")            = 괄호 안에 있는 가지고 오고 싶은 아이디값

DOM을 이용한 요소 접근하기

        客體 ( = 객체)
            客 : 손(어떤 사람) 객
            體 : 몸 체
            JS에서는 HTML 요소를 객체라고 표현
        1. id로 접근하기 : document.getElementById("id속성값");
            -> 아이디가 일치하는 요소(객체)를 얻어옴

            *참고. id값 빼고 모두 뒤에 Name이 붙음

        2. class로 접근하기 : document.getElementsByClassName("class속성값");
            -> 클래스가 일치하는 모든 요소를 얻어와 배열 형태로 변환
            -> 단체로 클래스명이 적힌 모든 요소를 갖고온다는 개념이 존재

        3. name로 접근하기 : document.getElementsByName("name속성값");
            -> name이 일치하는 모든 요소를 얻어와 배열(NodeList, 유사배열)형태로 변환

        4. tag로 접근하기 : document.getElementsByTag("Tag속성값");
            -> tag가 일치하는 모든 요소를 얻어와 배열 형태로 변환

        5. CSS 선택자로 접근하기
            ① document.querySelector("CSS 선택자");
            ->선택자가 선택한 요소 중 첫 번째 요소를 반환
            
            ② document.querySelectorAll("CSS 선택자");
             ->선택한 요소 모두 배열(NodeList, 유사배열) 형태로 변환

        ★ NodeList, 유사배열이란?
        배열        = [ ] 순차적으로 값을 묶어 담는 것
        유사 배열   = 배열처럼 여러 요소들을 담고 있지만 진짜 배열은 아님
                    map filter와같은 배열 기능을 사용할수 없음
                    대신 forEach 문을 사용하여 접근 가능
