flex 레이아웃

        display : flex
        div안에 작성한 태그들을 유연하게 배치할 수 있는 속성
        제일 많이 쓰이는 기능
        수평이나 수직으로 정렬이 가능
        flex의 경우 div(=가로 한 줄 전체 차지하는 block)형태를 inline형태로 바꾸는 효과가 있음
        inline = 가로에서 원하는 만큼만 공간을 차지
            display:flex; 와 같이 자주 쓰이는 속성 존재
        1. 가로로 존재하는 여백을 설정해주는 방법
        1-1. justify-concent:center;
            flex 컨테이너 내의 자식태그를 가로방향으로 가운데 정렬
        1-2. justify-concent:space-around;
            flex 컨테이너 내에서 자식태그들 사이에 균등한 여백을 주어 배치

        2. 세로로 존재하는 여백을 설정해주는 방법
            align-items : center;
            세로 줄에서 가운데 정렬
            주로 flex-direction:row;와 함께 사용할 때 세로 가운데 정렬이 필요한 경우 사용
            text - align - items
            ┖가로  ┴ 세로  ┘
        3. 한 줄 에서 있을 수 있는 태그의 너비가 넘어가면 자동으로 줄바꿈이 되도록 설정
        flex-wrap : wrap;
            최대한 가로에서 세로로 줄바꿈이 되어서 태그가 넘어가지 않도록 설정
             너비가 넘으면 어쩔 수 없이 줄바꿈이 되는 것

flex 와 inline차이

특징 요소 배치 방식 크기 제어 정렬 옵션 응용 범위
inline 같은 줄에 나열되며 필요한 공간만 차지 높이, 너비 지정이 어려움 정렬 옵션이 제한적 텍스트 및 링크 요소 등에 사용
flex 가로, 세로 방향으로 유연하게 배치 크기, 여백 쉽게 제어 가능 다양한 정렬 옵션 지원 전체 레이아웃 설계 주로 사용
