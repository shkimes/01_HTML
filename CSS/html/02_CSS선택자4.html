        형제관계가 있는 요소중에서 특정 요소만을 선택하는 선택자
        *중요* -> 위치를 기준으로 구분

        A:first=child -> 형제들 중 첫 번째가 A가 맞다면 선택
        A:last-child  -> 형제들 중 마지막이 A가 맞다면 선택
        A:nth-child(n=숫자값)-> 형제들 중 n번째에 A가 맞다면 선택
        A:nth-last-child-> 형제들 중 뒤에서 n번째가 A가 맞다면 선택자
