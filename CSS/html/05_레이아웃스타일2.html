padding, margin, border의 방향성



    상 우 하 좌를 읽히지 못하겠을 때 사용하는 또 다른 방법
    또는 우나 좌 하 에다가 특정 크리값을 주고 싶을 때 사용

    (ㅁㅁㅁ = padding, margin, border을 나타냄)

    ㅁㅁㅁ-top : 크기 -> 위쪽방향 크기 지정
    ㅁㅁㅁ-button : 크기 -> 아래쪽 방향 크기 지정
    ㅁㅁㅁ-left : 크기 -> 왼쪽 방향 크기 지정
    ㅁㅁㅁ-right : 크기 -> 오른쪽 방향 크기 지정

    eks, border의 속성 값은 [크기 모양 색상] 형식으로 작성

예제 활용하기

margin : auto

    -다른 요소와의 간격을 자동으로 지정해서 가운데 정렬하세 만드는 속성 값
    -기본적으로 좌우간격만 자동으로 지정
    -지정되는 대상은 block형식의 요소여아한다.(요소는 태그보다 큰 개념)
    inline, inline-block은 text-align:center

    text-align:center;
    1. inline태그 : <span<, strong, em, img등등
    2. inline-block태그 : button, input, img등등

    margin:auto;
    1. block 태그 : div, h1~h6, p, ul, ol, li, section등등
