        하이퍼링크태그
        하이퍼링크태그

        a태그 내부에 속성으로 href 작성하면 글자 색상과 밑줄이 생김
        글자색상은 기본적으로 파란색과 보라색.
        파란색 = 아직 클릭하지 않은 링크 색상
        보라색 = 이미 클릭해서 방문한 링크 색상

        a태그는 사용자가 특정행동을 했을 경우를 3가지 경우로 나누어 설정

        a:hover = 마우스를 링크 위에 올렸을 때 스타일
                주로 text-decoration : underline 밑줄 추가
        
        a:active = 마우스를 클릭했을 때 활성화 되는 스타일이
                주로 글자색 또 한 번 변경하고 클릭시 약간 줄어들거나 커지게 만듬

        a:visited = 사용자가 이미 방문한 링크에 스타일을 지정할 때 사용
                주로 color만 사용

        button input type = "submit" 와 a태그의 차이점
        button input type = "submit" 태그는 제출을 위한 용도로 주로 사용
        a href 태그는 특정 주소나 링크로 이동하기 위해 사용
