Transition 와 Transform

transition 속성

        transition 
        : 요소의 속성이 변경될 때 그 변화가 즉각적으로 나타나는 대신, 부드립게 전환되도록 애니메이션 효과 적용
        transition 
        :속성명 전환시간 전환 속도 곡선 전환이 시작되기 전 지연시간 대기했다가 변화
        예를 들어 배경색을 빨간색으로 변경할 경우 => 3초 후에 빨간색으로 변경시작!
        = transition : background-color 0.5s linear 3s;

transform 속성

        transform : 요소를 이동, 회전, 크기 조절, 뒤집기 등의 변환을 적용하는 데 사용
        1. transform : 요소를 x축y축 방향으로 이동
                transform  : translate(10px, 20px); => 가로 오른쪽으로 10px 세로 아래쪽으로 20px 이동
                transform  : rotate(45deg) =>
