태그 선택자

        -현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자
        [작성법]
        
            태그명{ 
                css 코드;

            }

id(#) 태그 선택자

        HTML 문서 내에서
        ID가 일치하는 요소를 선택하는 선택자
        [작성법]
        
            #id이름 {
                css코드;
            }

class(.) 선택자

        -html문서 내에서
        class 속성 값이일치하는 모든 요소를 선택하는 선택자
        -style에서는 class를 주로 사용
        -id 값은 주로 Java script에서 사용
        -class 를 다지인에선 필수로 사용하기
            (특히나 마지막은 개발자들과의 암묵적인 약속)
            
                .class이름명{
                    class 코드;
                }
