입력관련태그 활용편

select - option 태그

            select 태그
            - 사용자가 미리 정의된 옵션 중 하나를 선택할 수 있도록 드롭다운 목록을 생성하는 태그

            option 태그
            - 무조건 select 태그 내에 존재한다.
              개발자가 정의된 옵션을 지정해주는 태그

예제 코드

fieldset - legend 태그

            fieldset 태그
            -여러 개의 input 과 같은 입력 태그를 그룹화 시키는 태그
             form 태그 안에서 관련된 항목끼리 그룹화 할 때 많이 사용

            legend 태그
            - 무조건(높은 사용률로) fieldset 태그 내에 존재
            -fieldset 태그의 제목을 나타내는 태그 
            -fieldset 태그의 포함된 태그들의 그룹을 설명

예제 코드

필수 입력 사항

개인정보 항목 동의

개인정보 이용 약관(주)
