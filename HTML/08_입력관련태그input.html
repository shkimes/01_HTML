입력관련태그 input편

        input type 종류
        1. 글자           = text
        2. 비밀번호       = password (●로 보이게 함)
        3. 이메일         = email
        4. 전화번호        = tel
        5.숫자 및 범위     = number (숫자만 입력 가능)
        6. 범위           = range (슬라이더)
        7. 파일           = file (파일 올리기)
        8. 시간/주/월     = time/week/month
        9. 체크박스       = checkbox (중복 선택 가능)
        10. 라디오        = radio (하나만 선택 가능)
        11. 제출/리셋/숨김 = submit/reset/hidden
        12. 미리보기      = iframe
