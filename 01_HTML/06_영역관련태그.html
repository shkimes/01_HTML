영역관련태그

div 태그

        -문서(Document) 코딩(html) 내에서 구역이나 컨테이너를 지정하는데 사용
        - P 태그와 다른점 : P태그는 글자만 구역을 잡음
        -DIV태그는 그림 동영상 등 여러가지를 태그 틀 안에 담을 수 있음
        -div 태개를 나가면 자동으로 줄바꿈(enter)처리가 됨

span 태그

        -이어서 작성 가능 틀을 제공하는 태그 (div는 자동으로 줄바꿈처리 되는것이 차이점)

div와 span 비교하기

첫 번째 div
두 번째 div
세 번째 div
첫 번째 span 두 번째 span 세 번째 span

block 형식

  -화면 공간을 수직 분할(세로 화면을 가로로 잘라서 층을 나누는 행위)
  -가로 한 줄 전체를 차지하기 때문에 />태그를 닫는 순간 enter 줄바꿈이 이루어짐
  -width, height와 같은 크기 지정 속성 사용 가능
  
  -block 형식의 태그
  h1~h6       = 제목
  p/pre       = 문단
  hr          = 수평선
  div         = 영역 분할
  시맨틱 태그  = div 태그에 이름을 넣는 태그

inline 형식 태그

  -화면을 수평 분할
  -가로에서 큰 공간을 차지하지 않고 사용하고 싶은 만큼만 공간을 차지한 다음에 옆자리를 내어줌
  -width, height와 같은 크기 지정 속성 사용 불가 (중요)

  -inline 형식의 태그
  b      = 굵게
  strong = 굵게(청각장애인 편의있음)
  i      =  기울이기
  mark   = 글배경 색칠하기
  span   = 영역 분할
